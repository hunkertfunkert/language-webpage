UX, UI, and Web Design


UX

UX refers to user experience, the experience any person may have when interracting with a site. UX design specifically is the process in creating a good user experience through good web design choices. User experience is very important when designing a website. You could have the most useful website or best product in the world but if it is unappealing to the user then it may as well serve no purpose.



UI

UI refers to user interface and is essentially the background, links, buttons, images, and any other element within the webpage. UX is entirely dependent on UI, but the developer of a website only has influence on the UI to provide a good user experience. Good UI is important to provide a smooth and pleasing experience to any visitor of a website. Care should be taken to use CSS and often JavaScript to make the UI as good as possible to fit the website specifications. This website is an example of bad UI.



Web Design

Web Design is a broad term describing the process in which a website is created and upheld. Careers in web design can vary a lot, but are often split into front end and back end web developers. A front end web developer creates the visuals and everything a user will see when visiting a site, while a back end developer works on functionality and server related aspects of the website.
